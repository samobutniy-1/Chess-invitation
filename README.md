# ♟️ Chess Club Landing Page

A simple landing page for a chess club, built as a practice project for learning **Tailwind CSS**, **Framer Motion animations**, and **basic HTML form handling**.

## 🛠️ Tech Stack

- **React** — UI
- **Tailwind CSS** — styling
- **Framer Motion** — animations

## ✨ Features

- Smooth scroll animations with `whileInView`
- Staggered list animations with `staggerChildren`
- Responsive layout for mobile and desktop
- Sign up form with built-in HTML validation (no backend)

## 🚀 Getting Started
```bash
npm install
npm run dev
```
